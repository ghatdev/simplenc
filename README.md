# simplenc
## ENG description
This is really simple nc alternative.  
Just use like this:
- Linux/Mac (Bash-like environment)
```bash
$ simplenc [HOST] [PORT]
```
- Windows (CMD-like environment)
```
> simplenc [HOST] [PORT]
```
